体能教练课表系统 - 使用说明
=============================

一、启动项目
-----------
cd /Users/lu/fitness_schedule
uv run uvicorn main:app --host 0.0.0.0 --port 8000

启动后访问: http://localhost:8000


二、关闭项目
-----------
按 Ctrl+C 停止服务

或者运行:
pkill -f uvicorn


三、主要功能
-----------
1. 注册/登录 - 支持多教练账号
2. 学员管理 - 新增、编辑、删除学员
3. 课程管理 - 设置课程名称和时长（30/60/90分钟）
4. 课表日历 - 月视图和周视图切换
5. 拖拽调整 - 课程可直接拖拽到其他日期/时间
6. 快速添加 - 点击日期滑出时段面板，快速选择时段


四、初始设置
-----------
首次使用需要:
1. 注册教练账号
2. 添加课程（如：体能训练、力量训练等）
3. 添加学员信息
4. 在日历上点击日期创建课程安排
